# рџ“‚ Penjelasan Folder

| Folder | Fungsi |
|---------|--------|
| `app/` | Halaman aplikasi menggunakan Next.js App Router |
| `app/api/` | Seluruh REST API aplikasi |
| `app/(dashboard)/` | Dashboard Admin |
| `app/[username]/` | Halaman CV publik berdasarkan username |
| `components/` | Komponen UI yang dapat digunakan kembali |
| `components/ui/` | Komponen dasar seperti Button, Input, Card, Modal |
| `lib/` | Konfigurasi Prisma, autentikasi, helper, validasi, dan utilitas |
| `hooks/` | Custom React Hooks |
| `services/` | Logika bisnis dan komunikasi dengan database |
| `prisma/` | Schema, migrasi, dan seeding database |
| `public/` | Asset statis (gambar, ikon, logo) |
| `app/globals.css` | File CSS global |
| `docs/` | Dokumentasi proyek |
| `middleware.ts` | Middleware autentikasi dan proteksi halaman |
| `.env` | Konfigurasi environment lokal |
| `README.md` | Dokumentasi utama proyek |
