# Authentication

Semua endpoint Dashboard memerlukan login.

Header

```
Authorization: Bearer <JWT Token>
```

Endpoint Public tidak memerlukan login.

---

# Response Format

## Success

```json
{
  "success": true,
  "message": "Data berhasil diproses",
  "data": {}
}
```

## Error

```json
{
  "success": false,
  "message": "Terjadi kesalahan"
}
```

| Code | Keterangan |

# 1. Authentication

## Login

### Endpoint

```
POST /auth/login
```

### Request

```json
{
  "email": "admin@email.com",
  "password": "password123"
}
```

### Response

```json
{
  "success": true,
  "message": "Login berhasil",
  "data": {
    "token": "JWT_TOKEN"
  }
}

---

## Register

### Endpoint

```
POST /auth/register
```

### Request

```json
{
  "email": "user@email.com",
  "username": "username",
  "password": "password123"
}
```

### Response

```json
{
  "success": true,
  "message": "Registrasi berhasil"
}
```

---

## Logout

### Endpoint

```
POST /auth/logout
```

### Response

```json
{
  "success": true,
  "message": "Logout berhasil"
}
```

# Catatan Pengembangan

- Backend menggunakan Next.js App Router (`app/api`).
- Database menggunakan Neon PostgreSQL.
- ORM menggunakan Prisma.
- Endpoint Dashboard hanya dapat diakses setelah login.
- Endpoint Public dapat diakses tanpa autentikasi.
- Semua request dan response menggunakan format JSON.
- URL publik CV menggunakan format:

```
https://domain.com/{username}
```

Contoh:

```
https://cvonline.vercel.app/dadangdudung
```
